Config Options :


Mandatory as said, compulsory

colModel : [ {index : '<index of the column in data>'
,colName : '<column name for the display grid>'}
,{...}
,...
]

url : '<url to fetch the data>'

Optional has defaults

headerTitle : '<title for the grid to be displayed in header>'
default: '/' 
openFileFn : function(){
	// code todo with the single file
}
default: {} 
parentID : '<value to be sent to server to fetch initial files and folders>'
default: 0 
param : '<url-parameter to be sent to server for fetching data>'
default: 'id' 
imageCfg : {cfg : '<colname on wich images depend; to say, the type>',
images : [{type : '<name of the column type>'
,image : '<image path>'}
, {...}
, ...
]
}
default: ''

Usage :


HTML simple div tag

<div class="listing>
<div>

JS simple plugin call, with mandatory options

$('.listing').folderGrid({
	colModel: [
			{index : 'name', colName : 'Name'}
			,{index : 'size', colName : 'Size'}
			,{index : 'type', colName : 'Type'}
			  ]
	,url:'getData'
});

DATA simple JSONArray

[
{'id' : '2', 'isFolder' : 'true', 'name' : 'Folder1', 'size' : '', 'type' : 'folder', 'modified' : '4/6/2012'},
{'id' : '3', 'isFolder' : 'false', 'name' : 'File1', 'size' : '4KB', 'type' : 'file', 'modified' : '4/6/2012'},
{'id' : '4', 'isFolder' : 'false', 'name' : 'File2', 'size' : '5KB', 'type' : 'file', 'modified' : '4/6/2012'}
]

                {index : 'size', colName : 'Size'},

				,{type : 'folder', image : 'images/folder.gif'}
